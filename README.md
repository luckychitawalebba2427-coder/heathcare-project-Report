# 🏥 Healthcare Analytics Dashboard

An interactive Power BI dashboard designed to analyze healthcare data and provide actionable insights for better decision-making. The dashboard enables users to explore patient demographics, medical conditions, admission trends, billing information, insurance coverage, and healthcare performance metrics through dynamic visualizations and filters.

## 📊 Dashboard Features

* Interactive slicers and filters
* Patient demographics analysis
* Medical condition distribution
* Admission type analysis
* Insurance provider insights
* Billing and revenue analytics
* Emergency case monitoring
* Year-wise patient trends
* KPI cards for key healthcare metrics

## 🛠️ Technologies Used

* Power BI Desktop
* Power Query
* DAX (Data Analysis Expressions)
* Data Modeling

## 📈 Key Metrics

* Total Patients
* Average Billing Amount
* Female & Male Patient Percentage
* Emergency Cases
* Top Medical Condition
* Insurance Provider Analysis
* Test Result Distribution

## 🎯 Project Objective

The objective of this project is to transform raw healthcare data into meaningful insights through interactive visualizations, helping healthcare organizations monitor performance, identify trends, and support data-driven decision-making.
